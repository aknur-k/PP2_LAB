# Mickey's Clock Application

## 📌 Description
This is a digital-style clock application built using Pygame.  
It displays the current system time using Mickey Mouse-style hands instead of normal clock hands.

## 🕒 Features
- Displays current minutes and seconds
- Uses rotating images as clock hands
- Right hand → minutes
- Left hand → seconds
- Updates in real time

## ⚙️ Requirements
- Python 3.12
- Pygame library
